# LLM Coding Challenge Shootout

This project is inspired by and builds upon Luke Skyward's blog post ["I Built 3 Apps with DeepSeek, OpenAI o1, and Gemini â€” Here's What Performed Best"](https://medium.com/@datasciencedisciple/i-built-3-apps-with-deepseek-openai-o1-and-gemini-heres-what-performed-best-065cb460d761).

## Overview

This repository contains a collection of coding challenges, prompts, and applications built using various Large Language Models (LLMs). The project aims to explore and compare the capabilities of different AI models in real-world coding scenarios.
